# Dark Eden Digital

## Overview
Dark Eden Digital is a Godot 4 project that digitizes the legacy Dark Eden card game. The current focus is on automating the migration of the legacy card database from Magic Set Editor (MSE) files into a structured JSON format, standardizing resources and mechanics, and building the foundational systems for visual UI development in Godot.

## Roadmap

- [x] Initial project setup (Godot 4)
- [x] Create custom parser for MSE file extraction (`parse_mse.py`)
- [ ] Parse and clean card data into structured JSON
- [ ] Standardize resources and mechanics into a data-driven structure
- [ ] Develop Godot importer for the parsed JSON (`mse_importer.gd`)
- [ ] Prepare codebase for visual UI development
- [ ] Implement foundational UI components in Godot
- [ ] Integrate parsed card data into UI screens
- [ ] Implement core game logic and mechanics
